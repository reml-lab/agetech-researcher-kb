Search Help

The knowledge base can be searched in the following ways:
  • Search by Name: Returns all researchers with names containing the specified search term.
  • Search by Keyword: Returns all researchers with research summaries containing the specified search term.
  • Search by Location: Returns all researchers with locations (city, region, or country) containing the specified search term.
  • Search by Proximity: Returns the ten researchers who are closest to the location specified in the search term. The search term can be a street address (e.g., 300 Massachusetts Ave, Amherst, MA), a partial address (e.g, Amherst, MA), or the name of a place (e.g, UMass Amherst).

MassAITC AgeTech Researcher Knowledge Base

The MassAITC AgeTech Researcher Knowledge Base provides functions for browsing, searching and viewing information about AgeTech researchers. Researcher profiles were constructed using data from OpenAlex and OrcID. Researcher profiles include research affiliations, top research topics, top co-authors, and a listing of related research papers. Each profile also includes a generative AI research summary based on the researcher's five most recent AgeTech papers and five most cited AgeTech papers. Profiles are searchable by name, keyword, and location.
